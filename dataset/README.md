# Dataset Format

Put images in these folders:

```text
dataset/train/biodegradable
dataset/train/non_biodegradable
dataset/validation/biodegradable
dataset/validation/non_biodegradable
```

Recommended split:

- 80% images in train
- 20% images in validation

Example:

If you have 100 biodegradable images:

- 80 images -> dataset/train/biodegradable
- 20 images -> dataset/validation/biodegradable
